Automated Pedestrian
Collision Avoidance System





APCA Presentation


Vehicles are becoming more and more automated. Already we can see systems on vehicles that aid drivers in parallel parking, reversing, and more. A future goal of the automotive industry is to have full automated cars. The Automated Pedestrian Collision Avoidance System is an essential piece to making that dream a reality.
The project our team is working on is Automated Pedestrian Collision Avoidance System (APCA). The project focuses on automatically braking when pedestrians are detected in front of the vehicle. The sensor should be able to detect a pedestrian whether it is in motion or not. Also, the system will track pedestrians and determine if braking is needed. Braking is applied once a pedestrian steps in front of the car. The top priority of the project is to avoid collisions with pedestrians. The secondary objective is to reduce time loss because of braking. This means the car should not be decelerated if the pedestrian is not in danger.
